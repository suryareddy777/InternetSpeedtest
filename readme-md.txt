# Internet Speed Test

A lightweight web application to test your internet connection speed. Built with Python (Flask) and JavaScript.

## Features

- Ping test
- Download speed test
- Upload speed test
- Historical data tracking
- Visual data representation with gauges and charts

## Requirements

- Python 3.7+
- Flask
- Gunicorn (for production deployment)

## Installation

1. Clone this repository:
   ```
   git clone https://github.com/yourusername/internet-speed-test.git
   cd internet-speed-test
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

3. Run the application locally:
   ```
   python app.py
   ```

4. Access the application at http://localhost:5000

## Deployment

This application is ready to be deployed on Render.com.
